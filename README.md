# Food Delivery Data Analysis Hackathon

This repository contains my hackathon submission for food delivery data analysis.

## Files
- Food_Delivery_Data_Analysis.ipynb – Jupyter notebook with full data integration and analysis
- final_food_delivery_dataset.csv – Final merged dataset used for analysis

## Tools Used
- Python
- Pandas
- Google Colab

## Overview
The project combines data from CSV, JSON, and SQL files and analyzes order trends, user behavior, city and cuisine performance, membership impact, and revenue patterns.
